Project directory structure for an analytics project

2016 by John Paul Bida

Standardizing a directory structure for your analytics scripts and small projects makes sharing and exploring a teams code base significantly easier. In the analytics repository we use the directory structure outlined below. 

 
pXXXXX/
	README
	data
	rpgms
	results
		html
		json
		images
		tables
	docs

  • pXXXXX - Main project directory. The XXXXX is a five digit number that is unique to each project.
  • README - file describing the project using a standard template
  • data - Contains any raw data files, SQL queries, etc.
  • rpgms - Contains all R scripts
  • results - Contains all output generated by R scripts
  • docs - Contains any documentation related to the project.
